# Getting Started

Clone this repo

```
git clone "Repo Link"
```

Change directory to BD-Tour-Client

```
cd Task-Stead-Fast-React
```

Install the dependencies

```
npm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
